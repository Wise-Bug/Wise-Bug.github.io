Ubuntu18.04.1安装IntelliJ IDEA

下载地址：

https://www.jetbrains.com/idea/download/#section=linux

文件名：ideaIC-2019.2.4.tar.gz

1.切换到下载目录

2.解压

解压到 /opt 下：

sudo tar -zxvf ideaIC-2019.2.4.tar.gz -C /opt

3.进入到IDEA文件夹下的bin目录

cd /opt/idea-IC-192.7142.36/bin

4.启动 IDEA

./idea.sh

5.初次配置

一路next就好

然后退出重启就完成了。
