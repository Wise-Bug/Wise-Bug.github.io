Ubuntu 常用指令

Author: Wise-Ant Date: 八月 21, 2019  2:43:49 Category: 速查手册

键盘快捷键

Ubuntu-18.04 键盘快捷键可以自定义更改，更改路径：设置 → 设备 → 键盘，为了以后方便，请尽量更改为以下标准。

  • 打开终端 ：Ctrl + Alt + T （系统默认）

  • 打开 Bash 主页 ：Super(win键) （系统默认）

  • 在运行程序之间进行切换 ：Ctrl（Super）+ Tab （系统默认）

  • 中英文切换 ：Ctrl （需下载“搜狗输入法”自行配置）

  • 一般状态下的复制 ：Ctrl + c（系统默认）

  • 一般状态下的粘贴 ：Ctrl + v（系统默认）

  • 终端页面下的复制 ：Shift + Ctrl + c（系统默认）

  • 终端页面下的粘贴 ：Shift + Ctrl + v（系统默认）

  • 屏幕截屏 ：Ctrl + Alt + a （需要自行配制）

  • 屏幕录屏，启动后屏幕右上角有橙色标记 ：Shift + Ctrl + Alt + r（系统默认）

  • 打开Dash 应用栏目 ：Super + a （系统默认）

  • 切换到首个虚拟终端 ：Ctrl + Alt + F1 （系统默认）

  • 选择不同的虚拟终端 ：Ctrl + Alt + F3(F4)(F5) （系统默认）

  • 调整当前窗口大小 ：Super + 方向键 （系统默认）

  • 隐藏当前窗口 ：Super + H （系统默认）

  • 删除当前窗口 ：Alt + F4 （系统默认）

  • 恢复Chrome 浏览器上次所删除的所有页面 ：Shift + Ctrl + T（系统默认）

  • 锁屏：Super + L

  • 切换工作区域：Ctrl + Alt + 箭头

终端指令

当不清楚某个具体指令的用法时，可以使用：具体相关指令 –help ；进行查询

  • 打开终端：Ctrl + Alt + T

  • 退出某种模式/关闭终端：Ctrl + d

  • 显示本机基本配置：neofetch

  • 切换到 root 用户： sudo -i

  • 清屏：Ctrl + l

  • 停止命令：Ctrl + z ；还原：fg

  • 显示当前所属目录： pwd

  • 查看当前文件下的目录：ls ； -a 显示隐藏文件； -l显示文件详细信息

  • 创建目录：mkdir 目录名 ；默认当前路径下；(MaKeDIRectory)

  • 切换目录：cd 路径 ；可填全局路径或相对路径；(ChangeDirectory)

  • 切换到上级目录：cd ..

  • 切换至上一次所在的目录：cd -

  • 复制文件：cp 源文件路径 目标路径 （-f 强制复制，如有相同覆盖）

  • 复制目录：cp -r 目录路径 目标路径

  • 删除文件：rm 文件路径及文件名

  • 删除目录：rm -rf 目录路径

  • 重命名文件：mv 文件路径 文件路径及新名字 ；例如：mv ddd/zzz ddd/aaa ；

  • 查找文件/目录：locate string

  • 显示文件系统空间信息：df

  • 显示系统网络信息：ifconfig

  • 关机：sudo shutdown

  • 重启：sudo reboot

  • 查看当前存在哪些进程：ps -A

  • 结束某些进行：kill KID(进程号)

  • 添加新的普通用户：sudo adduser 新用户账号

  • 设置root用户密码：sudo passwd root

  • 显示文件内容：cat 文件名称

  • 显示文件前两行内容：head -n 2 文件名称

  • 显示文件后两行内容：tail -n 2 文件名称

  • 查找指定路径内满足字符串匹配的文件和目录：find 路径 -name 字符串

软件更新

  • 更新软件列表：sudo apt-get update

  • 更新软件：sudo apt-get upgrade

  • 安装软件：sudo apt-get install 软件名称

  • 卸载软件：sudo apt-get remove 软件名称

  • 卸载软件并删除配置文件：sudo apt-get purge 软件名称

  • 查看所有安装的软件：dpkg -l

  • 查看某个软件是否被安装：dpkg -l | grep 软件名称

压缩和解压文件命令

  • tar文件

    解包：tar zxvf filename.tar

    压缩：tar czvf filename.tar dirname

  • gz文件

    解压1：gunzip filename.gz

    解压2：gzip -d filename.gz

    压缩：gzip filename

  • .tar.gz 和 .tgz文件

    解压：tar zxvf filename.tar.gz

    压缩：tar zcvf filename.tar.gz dirname

    压缩多个文件：tar zcvf filename.tar.gz dirname1 dirname2 dirname3…..

  • bz2命令

    解压1：bzip2 -d filename.bz2

    解压2：bunzip2 filename.bz2

    压缩：bzip2 -z filename

  • .tar.bz2文件

    解压：tar jxvf filename.tar.bz2

    压缩：tar jcvf filename.tar.bz2 dirname

  • bz文件

    解压1：bzip2 -d filename.bz

    解压2：bunzip2 filename.bz

  • tar.bz文件

    解压：tar jxvf filename.tar.bz

  • z文件

    解压：uncompress filename.z

    压缩：compress filename

  • zip文件

    解压：unzip filename.zip

    压缩：zip filename.zip dirname
