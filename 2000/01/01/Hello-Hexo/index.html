Hello Hexo

Author: Wise-Ant Date: 一月 1, 2000  0:00:00 Category: 速查手册

保证 _posts 文件夹不为空。

查看是否可以输出图片。

Hexo 常用的几个命令：

1. hexo s ：

hexo s 是 hexo server 的简写，命令效果一致。

启动本地服务，用于预览主题。

默认地址：http://localhost:4000/

2. hexo n ：

hexo n 是 hexo new 的简写，命令效果一致。

该命令的主要作用是创建文章，例如：hexo n “Hello Hexo”

在配置好显示图片设置之后，会在 _posts 文件夹下新增两个文件，一个是同名的 .md 文件，一个同名的文件夹，文件夹用来放图片。

3. hexo g ：

hexo g 是 hexo generate 的简写，命令效果一致。

用来生成网址的静态文件，默认放在 public 文件夹中。

4. hexo clean ：

用来清除缓存文件 db.json 和 已生成的静态文件 public。

一般在执行 hexo g ，前先执行 hexo clean 比较稳妥一点。

5. hexo d ：

hexo d是 hexo deploy的简写，命令效果一致。

用来将本地博客上传到 Github 中。

6. hexo new page ：

标准用法 hexo new page 网页名称

默认链接地址为： 主页地址/网页名称

网页不会出现在首页文章列表和归档中，也不支持设置分类和标签。
